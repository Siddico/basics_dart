# Programming Basics: Loops & Functions
This repository contains simple examples that explain two fundamental programming concepts:

* **Loops**
* **Functions**

The goal of this repository is to help beginners understand how these concepts work through clear and practical examples.

## Repository Structure
.
├── loops
└── functions

### Loops
This file contains examples of different loop types used in programming, such as:

* for loops
* while loops
* simple iteration examples

These examples demonstrate how to repeat actions efficiently in code.

### Functions
This file contains examples of functions and how they are used to:

* organize code
* reuse logic
* make programs easier to read and maintain

Examples include defining functions, calling functions, and passing parameters.

## Purpose
This repository is meant for learning and practicing basic programming concepts that are essential in almost every programming language.
